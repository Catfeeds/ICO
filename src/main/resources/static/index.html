众筹萌芽

众筹兴起的时候，国内涌现许多垂直类众筹平台，一类是公益活动、门票等方面；另一类则是针对中小企业提供创业融资服务的股权众筹平台。

巨头涌入

随着京东、阿里等巨头涌入到了众筹领域，国内的众筹市场格局也迎来了2.0时代。很多垂直类众筹都开始纷纷转型谋发展。

迈向新时代
